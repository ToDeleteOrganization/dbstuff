<p> </p>
<p>BCM Storage Service was designed to support high performance for full BCM documents handling. In order to achieve high performance, BCM Documents are not serialized/deserialized, but they are saved directly as String. BCM Storage Rest endpoints are deployed under path /bcm</p>
<ul>
  <li>
    <strong>AddDocumentResponse addFullDocument(DocumentFile doc)</strong>
    <br/>
    <ul>
      <li>add a new full Document to BCM Service storage. Paragraph units and file skeleton are included too.</li>
      <li>Request method: POST</li>
      <li>Example: <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">http://{bcm-server}:8080/bcms/bcm/fullDocument</a>
      </li>
      <li>Rest path: /fullDocument</li>
      <li>request MIME Type: application/json;charset=UTF-8</li>
    </ul>
  </li>
  <li>
    <strong>String getFullDocument(String uuid)</strong>
    <br/>
    <ul>
      <li>retrieves a document by document id as String. Documents retrieved through this endpoint will have skeleton and paragraph units populated </li>
      <li>Request method: GET</li>
      <li>Example: <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca">http://</a>
        <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">{bcm-server}</a>
        <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca">:8080/bcms/bcm/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca</a>
      </li>
      <li>Rest path: /fullDocument/{uuid}</li>
      <li>response MIME Type: text/plain;charset=UTF-8</li>
    </ul>
  </li>
  <li>
    <strong>String updateFullDocument(Document doc)</strong>
    <br/>
    <ul>
      <li>updates a Document, including skeleton and paragraph units</li>
      <li>Request method: PUT</li>
      <li>Example: <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">http://</a>
        <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">{bcm-server}</a>
        <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">:8080/</a>
        <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca">bcms/bcm/</a>
        <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">fullDocument</a>
      </li>
      <li>Rest path: /fullDocument/{uuid}</li>
      <li>request MIME Type: application/json;charset=UTF-8</li>
    </ul>
  </li>
  <li>
    <strong>String deleteFullDocument(String uuid)</strong>
    <br/>
    <ul>
      <li>deletes all versions of the document</li>
      <li>Request method: GET</li>
      <li>Example: <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">http://</a>
        <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">{bcm-server}</a>
        <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">:8080/</a>
        <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca">bcms/bcm/</a>
        <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">deleteFullDocument</a>/<a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">4a5690b3-dccd-49b9-a59c-2ff2f23b96ca</a>
      </li>
      <li>Rest path: /deleteFullDocument/{uuid}</li>
    </ul>
  </li>
</ul>
<ul>
  <li>
    <strong>GetStorageDocumentsResponse getDocuments()</strong>
    <ul>
      <li>retrieves all existing documents in BCM Service storage</li>
      <li>Request method: GET</li>
      <li>Example: <a href="http://clujeditor01:8080/wse/lue/documents">http://</a>
        <a href="http://clujeditor01:8080/wse/lue/fullDocument/4a5690b3-dccd-49b9-a59c-2ff2f23b96ca" style="text-decoration: underline;">{bcm-server}</a>
        <a href="http://clujeditor01:8080/wse/lue/documents">:8080/bcms/bcm/documents</a>
      </li>
      <li>Rest path: /documents</li>
      <li>response MIME Type: application/json;charset=UTF-8</li>
    </ul>
  </li>
</ul>
